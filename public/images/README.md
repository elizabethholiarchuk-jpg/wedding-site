# Image Placeholders

This directory should contain your wedding photography images.

## Required Images

### Home Page Preview (4 images)
- `preview-1.jpg` - Portrait orientation (1200x1600px recommended)
- `preview-2.jpg` - Landscape orientation (1200x800px recommended)
- `preview-3.jpg` - Portrait orientation (1200x1600px recommended)
- `preview-4.jpg` - Landscape orientation (1200x800px recommended)

### Portfolio Page (12 images)
- `portfolio-1.jpg` through `portfolio-12.jpg`
- Alternate between portrait (1200x1600px) and landscape (1200x800px) orientations
- Images should be high-quality, edited wedding photography

## Image Guidelines

- **Format**: JPG or WebP
- **Size**: Optimize for web (under 500KB per image recommended)
- **Dimensions**: Maintain aspect ratios as specified above
- **Quality**: High-resolution, professionally edited
- **Style**: Editorial, minimal, consistent with brand aesthetic

## Optimization

Before adding images to this directory, optimize them using tools like:
- ImageOptim (Mac)
- TinyPNG (online)
- Squoosh (online)

This ensures fast page load times while maintaining visual quality.

## Temporary Placeholders

Until you add your actual photography, you can generate placeholder images using:
- https://placeholder.com
- https://unsplash.com (search for "wedding photography")
- Or use the generate_image tool to create custom placeholders
